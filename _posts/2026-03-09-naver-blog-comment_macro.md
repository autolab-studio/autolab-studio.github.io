# 네이버 블로그 자동 댓글 프로그램  
### (키워드 검색 기반 블로그 자동화 매크로)

블로그 마케팅을 하다 보면 반복적으로 해야 하는 작업이 많습니다.

예를 들어 이런 작업입니다.

- 특정 **키워드로 블로그 검색**
- 검색 결과에서 **특정 블로그 글 방문**
- 일정 시간 **체류**
- **공감(좋아요) 클릭**
- **댓글 작성**

이런 작업을 여러 계정으로 반복적으로 수행해야 하는 경우가 많습니다.

이번에 소개할 프로그램은 이러한 과정을 **자동으로 처리하는 네이버 블로그 자동화 프로그램**입니다.

---

# 프로그램 기능

이 프로그램은 다음과 같은 작업을 자동으로 수행합니다.

### 1️⃣ 네이버 로그인 자동화

엑셀 파일에 등록된 계정을 이용해  
여러 계정을 **순차적으로 로그인**합니다.

- 다계정 지원
- 프록시 설정 가능
- 자동 로그인

---

### 2️⃣ 키워드 기반 블로그 검색

지정한 키워드를 네이버에서 검색합니다.

예시
- 테니스 라켓 추천
- 맛집 후기
- 네이버 블로그 체류


검색 결과에서 블로그 글을 탐색합니다.

---

### 3️⃣ 특정 블로그 글 자동 방문

검색 결과에서 **지정된 블로그 글을 찾으면 자동으로 방문**합니다.

그리고 다음 작업을 수행합니다.

- 블로그 글 체류
- 스크롤 동작
- 사용자 행동처럼 자연스럽게 움직임

---

### 4️⃣ 블로그 체류 및 스크롤

단순 방문이 아니라 실제 사용자처럼 행동합니다.

예시 동작

- 랜덤 시간 체류
- 페이지 스크롤
- 상하 이동

이런 행동을 통해 **자연스러운 방문 패턴을 만들 수 있습니다.**

---

### 5️⃣ 공감(좋아요) 자동 클릭

블로그 글에 들어간 후

- 공감 버튼 클릭
- 이미 공감된 글은 자동 스킵

---

### 6️⃣ 댓글 자동 작성

미리 입력한 댓글 목록 중에서 **랜덤으로 댓글을 작성합니다.**

예시

- 좋은 글 잘 보고 갑니다 🙂
- 정보 감사합니다!
- 유익한 글이네요 👍


댓글은 프로그램에서 자동으로 선택됩니다.

---

# 프로그램 특징

이 프로그램은 일반적인 매크로와 달리 **실제 사용자 행동을 최대한 모방**하도록 제작되었습니다.

예를 들어

- 랜덤 체류 시간
- 랜덤 스크롤
- 다양한 댓글
- 여러 계정 순차 실행

이런 방식으로 작동합니다.

---

# 프로그램 사용 예시

이 프로그램은 다음과 같은 상황에서 활용할 수 있습니다.

- 블로그 마케팅 자동화
- 체류 / 공감 / 댓글 작업 자동화
- 블로그 활성도 관리
- 키워드 기반 블로그 방문

# 네이버 블로그 체류 자동화 프로그램  
### (키워드 검색 기반 블로그 방문 매크로)

블로그 마케팅이나 블로그 운영을 하다 보면  
다음과 같은 작업을 반복적으로 하게 됩니다.

- 특정 **키워드로 블로그 검색**
- 검색 결과에서 **특정 블로그 글 찾기**
- 해당 글 **방문**
- 일정 시간 **체류**

이러한 작업을 수십 번, 수백 번 반복해야 하는 경우도 많습니다.

이번에 소개하는 프로그램은  
이 과정을 **자동으로 수행하는 블로그 체류 자동화 프로그램**입니다.

---

# 프로그램 주요 기능

이 프로그램은 네이버 블로그 방문 과정을 자동화합니다.

### 1️⃣ 키워드 기반 블로그 검색

사용자가 입력한 키워드를 기준으로  
네이버에서 블로그 검색을 수행합니다.

예시
- 네이버 블로그 체류
- 맛집 후기
- 테니스 라켓 추천


검색 결과에서 블로그 글을 탐색합니다.

---

### 2️⃣ 특정 블로그 글 자동 탐색

검색 결과를 스크롤하며

지정한 블로그 주소가 포함된 글을 자동으로 찾습니다.

- 블로그 검색 결과 자동 스크롤
- 지정한 블로그 발견 시 자동 방문

---

### 3️⃣ 블로그 체류 자동화

블로그 글에 들어가면  
프로그램이 자동으로 체류를 수행합니다.

예시 행동

- 페이지 스크롤
- 랜덤 체류 시간
- 상하 방향 이동

실제 사용자가 글을 읽는 것처럼  
**자연스러운 방문 패턴을 생성합니다.**

---

### 4️⃣ 엑셀 기반 배치 작업

이 프로그램은 엑셀 파일을 통해  
여러 작업을 한 번에 실행할 수 있습니다.

예시 엑셀 구조

| keyword | blog_url |
|---|---|
| 네이버 블로그 체류 | https://blog.naver.com/example |
| 테니스 라켓 추천 | https://blog.naver.com/example2 |

이렇게 설정하면

- 키워드 검색
- 블로그 탐색
- 방문
- 체류

작업이 자동으로 실행됩니다.

---

# 프로그램 특징

이 프로그램은 단순한 매크로가 아니라  
**사람처럼 행동하도록 설계된 자동화 프로그램**입니다.

예를 들어

- 랜덤 스크롤
- 랜덤 체류 시간
- 자연스러운 페이지 이동

등을 통해 자동화 패턴을 최소화합니다.

---

# 추가 기능

이 프로그램은 다음 기능도 지원합니다.

### 프록시 지원

프록시를 설정하여  
다양한 IP 환경에서 작업을 수행할 수 있습니다.

---

### 테더링 IP 변경

ADB를 이용하여

- 모바일 테더링
- IP 변경

도 자동으로 수행할 수 있습니다.

---

# 활용 가능한 분야

이 프로그램은 다음과 같은 경우 활용할 수 있습니다.

- 블로그 마케팅
- 블로그 체류 관리
- 키워드 기반 블로그 방문 자동화
- 반복적인 블로그 작업 자동화

## 네이버 블로그 서로이웃추가 자동화 매크로 프로그램

네이버 블로그를 운영하다 보면  
**서로이웃 신청, 공감, 댓글 작성 등의 반복 작업**을 해야 하는 경우가 많습니다.

하지만 이러한 작업을 직접 하나씩 진행하면 많은 시간이 소요됩니다.

이 프로그램은 **네이버 블로그 검색 결과를 기반으로 다양한 활동을 자동으로 수행하는 자동화 프로그램**입니다.

검색 키워드를 입력하면 관련 블로그 게시글을 탐색하고 설정한 작업을 자동으로 진행합니다.

---

## 주요 기능

이 프로그램은 다음과 같은 작업을 자동으로 수행할 수 있습니다.

- 네이버 블로그 검색 결과 자동 탐색
- 블로그 게시글 공감 자동 클릭
- 서로이웃 신청 자동 진행
- 댓글 자동 작성
- 랜덤 딜레이 적용으로 자연스러운 작업 수행
- 작업 로그 확인 기능

사용자는 **검색 키워드와 작업 수만 설정하면 자동으로 블로그 활동을 진행할 수 있습니다.**

---

## 프로그램 특징

이 프로그램은 **Python 기반 자동화 프로그램**으로 제작되었습니다. :contentReference[oaicite:0]{index=0}

주요 특징은 다음과 같습니다.

- Selenium 기반 웹 자동화
- 네이버 블로그 모바일 페이지 활용
- 랜덤 딜레이 적용
- GUI 프로그램 제공
- 설정 저장 기능

또한 프로그램 실행 후 **작업 로그를 실시간으로 확인할 수 있어 작업 진행 상태를 쉽게 확인할 수 있습니다.**

---

## 프로그램 사용 방식

사용자는 프로그램 실행 후 다음 정보를 입력하면 됩니다.

- 작업할 블로그 수
- 검색 키워드
- 네이버 로그인 정보
- 댓글 내용
- 서로이웃 요청 메시지

이후 실행 버튼을 누르면 프로그램이 자동으로 블로그를 탐색하고 설정된 작업을 진행합니다.

---

## 활용 예시

이 프로그램은 다음과 같은 상황에서 활용할 수 있습니다.

- 블로그 운영 자동화
- 블로그 활동 관리
- 블로그 홍보 작업 자동화
- 반복적인 블로그 활동 자동 처리

특히 **블로그 운영 시간을 줄이고 싶은 사용자에게 유용한 프로그램**입니다.
